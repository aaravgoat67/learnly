Anonymous · Physics · Class 11
Why does current lag behind voltage in an inductor? I've read it 3 times and still don't get it 😭
🤖
Nova AI · Answered in 2 min
Think of it like a flywheel 🎡

When you push a flywheel (apply voltage), it doesn't spin instantly — it takes time to build momentum. An inductor is the same — it resists changes in current. So when voltage is applied, current builds up slowly behind it. That "lag" is 90° in a perfect inductor.

Key takeaway: Voltage leads, current follows. ELI = ICE 😄
